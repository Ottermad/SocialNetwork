Welcome to Social Network

By Charles Thomas & Thomas Peregrine

We are building a simple social network in Flask. Currently, we are nameless so please feel free to suggest one.

You can check out our project on Github here and we will hopefully have a live version up soon. If you are interested please feel free to fork the project and send us a pull request.

Features

Feature Description Status Branch
Instant Messaging We will include a way to instantly message other users. At the moment it is powered by AJAX but may be moved to websockets Implemented - subject to updates master
Posts Each user will be able to post public messages. Hopefully, this will integrate with the friendship system. Implemented - submit to implementation of friendship system master
Friendship Users will be able to friend each other - limiting whose posts they see. The system will be more similar to Twitter followers than Facebook. Not started friendship

Tools Used

Tool Description
Python Used to write the backend of our app.
Flask A Python micro web framework used to build our app.
Peewee A simple database ORM for our models.
HTML & Jinja2 When building the frontend of our app we used HTML with Jinja2 templating (which is included by default in Flask) to load data from the backend into our pages.
Sass & CSS & skeleton.css To style our site we used Sass to help speed up development as well as skeleton.css - a minimalistic frontend framework.
Javascript & JQuery To send and receive posts without reloading the page we used AJAX (or more accurately AJAJ).
