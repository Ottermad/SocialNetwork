Welcome to Social Network

By Charles Thomas & Thomas Peregrine

We are building a simple social network in Flask. Currently, we are nameless so please feel free to suggest one.

You can check out our project on Github here and we will hopefully have a live version up soon. If you are interested please feel free to fork the project and send us a pull request.

Design

The sites aesthetic design is supposed to feel clean and fairly modern. It takes inspiration from teamtreehouse.com. There is an emphasis on curves and light colours

Features

Feature Description Status Branch
Posts editor Posts are created via the use of markdown, we provide an easy to use editor for the posts. This also includes a live preveiw Finished - however it is likely to receive alterations. design, working
Instant Messaging We will include a way to instantly message other users. At the moment it is powered by AJAX but may be moved to websockets Implemented - subject to updates master
Posts Each user will be able to post public messages. Hopefully, this will integrate with the friendship system. Implemented - submit to implementation of friendship system. Ability to delete and edit posts on way. master
Friendship Users will be able to friend each other - limiting whose posts they see. The system will be similar to Facebook. Mostly implemented. Additional features on way. friendship
Syntax Highlighting Posts will receive syntax highlighting on code blocks, this will most likely be powered by highligh.js Planned, but not started design - probably

Tools Used

Tool Description URL
Python Used to write the backend of our app. https://www.python.org/
Flask A Python micro web framework used to build our app. http://flask.pocoo.org/
Peewee A simple database ORM for our models. http://jinja.pocoo.org/docs/dev/
HTML & Jinja2 When building the frontend of our app we used HTML with Jinja2 templating (which is included by default in Flask) to load data from the backend into our pages. https://peewee.readthedocs.org/en/latest/
CSS & font awesome We used CSS to style the site, we also used a few libary's and frameworks to speed up the process. Font awesome is a really useful framework for using icons in a webpage, they work as a font and some css styling. We also used normalize.css for cross platform consistency http://fortawesome.github.io/Font-Awesome/ https://github.com/necolas/normalize.css/
Javascript & JQuery To send and receive posts without reloading the page we used AJAX (or more accurately AJAJ). https://jquery.com/
PageDown Editor All of the posts are written in markdown, so to provide an editor for them we decided to use the pagedown editor, which also features in stackexchange.com. We decided to restyle the tool bar uisng icons from font-awesome https://code.google.com/p/pagedown/
